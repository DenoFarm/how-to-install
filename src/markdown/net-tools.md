# Ubuntu / Debian

---

## # Using apt

```bash
sudo apt update
sudo apt install net-tools
netstat -v
```

# CentOS / RHEL

---

## # Using yum

```bash
sudo yum update

netstat -v
```


# Arch Linux

## # Using pacman

```bash
pacman -S netstat-nat
netstat -v
```
