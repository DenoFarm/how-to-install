# Ubuntu / Debian / Mint

---

## # Using apt

```bash
sudo apt update
sudo apt install zip
zip -v
```

## # Using dnf

sudo dnf install zip

```

```

# Arch Linux

## # Using pacman

```bash
sudo pacman -S zip
zip -v
```
