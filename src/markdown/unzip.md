# Ubuntu / Debian / Mint

---

## # Using apt

```bash
sudo apt update
sudo apt install unzip
unzip -v
```

<br />

# RedHat / CentOS / Fedora

---

## # Using dnf

```bash
sudo dnf install unzip
unzip -v
```

<br />

# OpenSuse

---

## # Using zypper

```bash
zypper install unzip
unzip -v
```

<br />

# Arch Linux

---

## # Using pacman

```bash
sudo pacman -S unzip
unzip -v
```
