# Ubuntu / Debian

---

## # Using apt

```bash
sudo apt update # Refresh your local package index
sudo apt install git # Install Git
git --version # Check that the install was successful
```

<br />

# Fedora

---

## # Using dnf

```bash
sudo dnf install git # Install Git
git --version # Check that the install was successful
```

## # Using yum

```bash
sudo yum install git # Install Git
git --version # Check that the install was successful
```
